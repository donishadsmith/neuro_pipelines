Additional Postprocessing

BOLD data was converted to percent signal change [(signal − mean) / mean × 100], with the mean computed only on non-censored volumes. Data were then iteratively smoothed to a FWHM of {{ fwhm }} mm using AFNI 3dBlurToFWHM.

{% if analysis_type == "gPPI" %}

The seed timeseries was not smoothed to prevent the extracted timeseries from being a weighted average of the surrounding voxels. Furthermore, the seed region was extracted after the BOLD data was converted to percent signal change. The seed timeseries was denoised using the same nuisance parameters as the BOLD data; however, AFNI's 3dTproject was used to simply orthogonalize the seed to the nuisance regressors rather than prewhiten it, as temporal autocorrelation in the seed must be preserved for proper deconvolution. High-motion volumes were handled by using -cenmode ZERO, which replaces flagged volumes with zeros rather than removing or interpolating them. This retains the original timeseries length while preventing motion-contaminated signal or interpolation artifacts from propagating through the deconvolution and PPI interaction term.

Both the seed timeseries and the condition regressors were upsampled to {{ upsample_dt }} s (Masharipov, 2024). A binary condition vector (1 = condition active, 0 = condition inactive) was generated for each condition within the {{ task }} task (i.e., {{ condition_names | join(", ") }}). Each binary condition vector was mean centered so that the subsequent PPI interaction term would not be highly correlated with the main effect of the seed timeseries, which could otherwise result in spurious results that attribute shared variance with the seed to the interaction term (He, 2025). Prior to deconvolution, the upsampled seed timeseries was padded at both edges using reflect padding with a pad length of {{ pad_length }} samples (pad_seconds / upsample_dt) to minimize sharp boundary artifacts; the padding was removed immediately after deconvolution. {{ hrf_model_desc }} The seed timeseries was deconvolved using AFNI's 3dTfitter -FALTUNG with this impulse response function and the following penalty syntax: {{ faltung_penalty_syntax }}. Finally, the PPI interaction term was created by multiplying the deconvolved neural signal with the mean-centered binary condition vector (via 1deval) and convolving the product with the selected HRF model. The PPI interaction term was then downsampled back to the original TR grid of {{ tr }} s.

Model Specification

The design matrix was created with AFNI 3dDeconvolve and the model was estimated with 3dREMLfit to account for temporal autocorrelation in the residuals, producing more accurate standard errors and more valid t-statistics. The beta coefficients themselves are unaffected by the temporal autocorrelation correction.
