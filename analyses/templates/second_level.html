Nonparametric Analysis (PALM)

Group-level inference was performed using PALM (Permutation Analysis of Linear Models) with threshold-free cluster enhancement (TFCE). TFCE eliminates the need to select an arbitrary cluster-forming threshold by integrating over all possible thresholds, weighting each voxel by a combination of local signal height and spatial extent. Family-wise error rate was controlled across both contrast directions using the -corrcon flag.
